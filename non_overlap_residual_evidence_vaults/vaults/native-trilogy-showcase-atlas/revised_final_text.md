# Book Trilogy Atlas

## Public Text

The Book Trilogy Atlas is the structural companion to the showcase: it makes the manuscript's map visible by showing how books, chapters, concepts, and visual identities relate to each other.

The atlas converts the trilogy from a linear reading object into an inspectable architecture. A chapter can be understood by its book role, argument role, source role, neighboring chapters, and visual/storytelling function.

The work is strongest when it is judged by concrete components rather than by ambition alone. The current public object should therefore be read through these components:

- book-to-chapter map
- chapter relation surface
- argument role field
- source role field
- visual/storytelling role field
