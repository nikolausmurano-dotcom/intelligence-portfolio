# Book Trilogy Manuscript Governance System

## Public Text

The Book Trilogy project governs a seventy-two-chapter manuscript system across three linked books: democratic restraint in the use of power, the formation of the individual inside democratic life, and political repair after violence.

The manuscript is treated as a governed chapter system rather than a loose set of essays. Each chapter needs a public title, argument backbone, source burden, book-specific voice rule, visual identity, and handoff relation to adjacent chapters.

The work is strongest when it is judged by concrete components rather than by ambition alone. The current public object should therefore be read through these components:

- chapter dossier registry
- argument-backbone field
- source-state queue
- book-register constitution
- visual-title governance
- cross-chapter handoff map

## Current Status

The project has a public proof surface and source-traceability material. The present editorial task is to keep the public explanation clear, bounded, and specific while preserving the underlying manuscript or method architecture.
