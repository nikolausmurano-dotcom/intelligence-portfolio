# 13 snapshot workbooks; 11-phase build scaffold - AI-assisted methods

## Public Text

The analytic-technique compendium gathers structured intelligence methods into a governed editorial object that can support human analysis, AI-assisted analysis, and human-AI comparison.

The compendium is organized as a method library: techniques require purpose, use case, misuse warning, input requirement, expected output, and conditions under which the technique changes analytic judgment.

The work is strongest when it is judged by concrete components rather than by ambition alone. The current public object should therefore be read through these components:

- technique registry
- method-purpose field
- misuse warning
- input/output schema
- human-AI comparison hook
- revision tracker

## Current Status

The project has a public proof surface and source-traceability material. The present editorial task is to keep the public explanation clear, bounded, and specific while preserving the underlying manuscript or method architecture.
