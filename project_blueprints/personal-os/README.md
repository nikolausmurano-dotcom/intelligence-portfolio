# Personal OS — MuranoOS, a personal operating system, local workbench, and project foundation

## Public Pitch

**What it is.** A local personal operating system for project development, not a private life-management page.

**Distinctive mechanism.** Projects move through departments, rooms, subrooms, agents, commands, source records, task state, review surfaces, and release outputs while preserving local memory and upstream/downstream handoffs. The local build is grounded in a Tauri / React / Rust / SQLite desktop architecture that can host project workstations rather than only static notes.

**Why it matters.** It demonstrates operating-system thinking for intellectual work: serious projects need state, roles, instruments, review loops, continuity, and local software infrastructure, not scattered chats and folders.
