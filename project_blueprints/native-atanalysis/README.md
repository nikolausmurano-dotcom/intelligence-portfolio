# Analysis Tradecraft

## Public Pitch

**What it is.** Analysis Tradecraft is a method room for moving an intelligence question from assumptions to evidence, alternatives, uncertainty, confidence, and review.

**Distinctive mechanism.** It routes an analytic problem through assumption surfacing, alternative generation, evidence weighting, uncertainty language, confidence discipline, bias checks, and reviewer challenge.

**Why it matters.** It matters because tradecraft is the discipline that keeps analysis from becoming a polished opinion.
