# Script Atelier (v0)

## Public Pitch

**What it is.** This package is a handoff-grade design and implementation-planning bundle for the proposed Murano Script Atelier room.

**Distinctive mechanism.** It converts an archive or handoff into three inspectable decisions: what the work product is for, which component pieces carry the work, and what action would turn it into a live room, paper, method, or software instrument.

**Why it matters.** It preserves the distinctive design logic for Script Atelier (v0), so the portfolio can show the actual mechanism and status instead of reducing the project to a file-count or handoff label.
