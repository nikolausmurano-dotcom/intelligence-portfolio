# Applied History — lab for using historical case study to inform contemporary policy

## Public Pitch

**What it is.** An applied-history lab for using historical cases as disciplined analytic instruments.

**Distinctive mechanism.** It connects portfolio-native case selection, Belfer-style applied-history logic, Hoover-style historical working-group discipline, analogy limits, institutional context, decision points, and present-use constraints so historical comparison can become a bounded case-study memo rather than loose analogy.

**Why it matters.** It matters because history is useful only when the transfer conditions, institutional differences, decision points, and analogy limits are explicit.

## Blueprint

### Vision

An applied-history lab for using historical cases as disciplined analytic instruments.

### Mechanism

It connects portfolio-native case selection, Belfer-style applied-history logic, Hoover-style historical working-group discipline, analogy limits, institutional context, decision points, and present-use constraints so historical comparison can become a bounded case-study memo rather than loose analogy.

### Built components

The lab's public surface uses a cream-and-parchment editorial palette with the Belfer / Allison-Ferguson / Hoover source list visible on the surface. The page's structural scaffold is in place; case-study density continues to grow as new cases are worked under the methodology.

### Source use

Source use is organized by role: method authorities, domain evidence, technical background, implementation evidence, and claim limits. The page should show why each source matters; direct quotations and stronger factual claims still require exact passage anchors before publication.

### Plan

Build case-study depth — each new case worked under the methodology with its source documentation visible — and bind the framework references to specific Belfer and Hoover passages rather than gesturing at them. Long-term, the lab develops alongside the contemporary analytical pages, not separately from them.
