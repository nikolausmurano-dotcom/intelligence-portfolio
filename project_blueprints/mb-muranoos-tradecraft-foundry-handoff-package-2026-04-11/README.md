## Public Pitch

**What it is.** The package hands off a governed, architecture-first integration plan for building "Tradecraft Foundry / Observatory" — a MuranoOS workspace-scale domain (not a single room) that preserves a lossless SAT-modernization corpus, objectifies it into versioned tradecraft products, and runs those work products through provenance, AI-governance, synthetic-content, and calibration controls before export.
