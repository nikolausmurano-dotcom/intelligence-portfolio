# IC analyst portfolio build kit; skill taxonomy S01-S26

## Public Pitch

**What it is.** IC analyst portfolio build kit; skill taxonomy S01-S26 is a preserved project packet for portfolio build kit that converts Georgetown MPAI (Master of Professional Studies in Applied Intelligence) coursework — counterterrorism papers, cyber CTI briefs, CI risk assessments, geospatial products, major academic papers — into IC-style sanitized work samples suitable for IC analyst hiring.

**Distinctive mechanism.** It converts an archive or handoff into three inspectable decisions: what the work product is for, which component pieces carry the work, and what action would turn it into a live room, paper, method, or software instrument.

**Why it matters.** It functions as a professional-development control surface: roles, competencies, evidence, and next actions are organized so preparation can be audited rather than guessed.
