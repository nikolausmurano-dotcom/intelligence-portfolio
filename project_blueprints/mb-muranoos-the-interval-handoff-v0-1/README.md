# The Interval (remote violence)

## Public Pitch

**What it is.** The Interval is a MuranoOS room handoff for studying remote violence as a structure of separation between perception, command, vulnerability, and consequence.

**Distinctive mechanism.** It routes remote-violence analysis through distance, mediation, command chain, target vulnerability, perception gap, moral injury, and institutional responsibility.

**Why it matters.** It turns an abstract ethical problem into an inspectable analytical chamber for conflict, technology, perception, and accountability.
