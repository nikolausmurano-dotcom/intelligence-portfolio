# 13 snapshot workbooks; 11-phase build scaffold — AI-assisted methods

## Public Pitch

**What it is.** 13 snapshot workbooks; 11-phase build scaffold — AI-assisted methods is a method packet for a version-history archive of the IC Analytic Technique Compendium as it grew from a v0.1 scaffold to a v1.3 eleven-phase compendium.

**Distinctive mechanism.** The mechanism is method operationalization: name the analytic technique, define its use condition, connect it to a case or judgment problem, record failure modes, and decide whether it is a live method page or a reference packet.

**Why it matters.** It turns analytic methods into reusable instruments with inputs, steps, failure modes, interpretation rules, and reviewer actions.
