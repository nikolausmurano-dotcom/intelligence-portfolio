# Cognitive Warfare — counter-intelligence doctrine, operationalized into analytical instruments

## Public Pitch

**What it is.** A combined counter-intelligence and cognitive-warfare doctrine page.

**Distinctive mechanism.** Each of the three doctrinal authorities is paired with its constructive operationalization — not "what does Olson say" but "what analytical move does Olson's principle enable and what failure mode does it catch." The operationalization is what turns the principles into instruments.
