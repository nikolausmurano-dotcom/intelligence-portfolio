# Lossless Study Room

## Public Pitch

**What it is.** Lossless Study Room is a MuranoOS handoff for Applied Intelligence Mission Control.

**Distinctive mechanism.** It combines distraction capture, per-course and cross-course Obsidian-style Zettelkasten structure, flashcard generation, source capture, coursework state, and review surfaces so graduate study becomes a governed knowledge-production room instead of scattered notes.

**Why it matters.** It matters because the same machinery that preserves coursework can later preserve research decisions, sources, and project continuity.
