# Canonical SAT dedup: 106 techniques normalized across 11 catalogues

## Public Pitch

**What it is.** Canonical SAT dedup is a method packet normalizing 106 structured analytic techniques across 11 catalogues.

**Distinctive mechanism.** It reconciles the Georgetown Registry, IC compendium, SAT and alternative-analysis traditions, red-team methods, CIA Tradecraft Primer, Army ATP, NATO alternative-analysis material, UK MOD red teaming, UFMCS Red Team Handbook, Morgan Jones, and normalized N-001 through N-106 records.

**Why it matters.** It matters because a SAT studio needs a deduplicated method canon before it can author, test, combine, or adapt techniques for human, AI, and human-AI workflows.
