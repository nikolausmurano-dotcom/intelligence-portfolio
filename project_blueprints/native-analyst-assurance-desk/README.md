# Analyst Assurance Desk

## Public Pitch

**What it is.** An analyst-assurance workstation for human review of AI-supported intelligence tasks.

**Distinctive mechanism.** It makes override decisions, analyst-model disagreement, task-family calibration, source review, structured analytic technique outputs, uncertainty, assumptions, reasoning paths, scoring rules, and release chronology visible as native objects.

**Why it matters.** It is meant to show what human-in-the-loop review actually requires: not trust in a reviewer, but surfaces that let the reviewer inspect sources, assumptions, uncertainty, reasoning paths, and machine-aided judgments before release.
