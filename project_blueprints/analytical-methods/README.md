# Structured Methods Library — method-composition page across Heuer, structured analytic techniques, and intelligence methodology

## Public Pitch

**What it is.** A general analytical-methodology library: the place where method families, source authorities, and reusable tradecraft moves are catalogued and made checkable.

**Distinctive mechanism.** Four mappings define every method in the catalogue: the question the method answers, the canonical source it derives from, the analytical trap it guards against, and the way the method composes with other methods.

**Why it matters.** It turns analytic methods into reusable instruments with inputs, steps, failure modes, interpretation rules, and reviewer actions.
