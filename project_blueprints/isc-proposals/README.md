# Counterintelligence Surveillance Framework — Beyond Surveillance Maximalism: a Smart Surveillance Scaling (S3) proposal

## Public Pitch

**What it is.** A counterintelligence policy proposal on smart surveillance scaling, not a generic surveillance page.

**Distinctive mechanism.** It critiques surveillance maximalism, specifies the Smart Surveillance Scaling framework, and links the proposal to counterintelligence tradecraft analogues and scaling constraints.

**Why it matters.** It demonstrates policy-design reasoning: the proposal has to say what kind of surveillance is useful, where scale becomes counterproductive, and how counterintelligence purpose changes the design.
