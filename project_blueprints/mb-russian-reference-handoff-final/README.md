# 100-item ranked bibliography with verification levels A/B/C x core/extended

## Public Pitch

**What it is.** Russian reference handoff is a ranked bibliography and source-authority packet for Russia and Eurasia work.

**Distinctive mechanism.** It classifies one hundred sources by verification level and core or extended role, including Routledge, Oxford, Cambridge Companion, Cambridge History, and Russia/Eurasia reference anchors.

**Why it matters.** It matters because Russia analysis should be built on ranked source authority rather than opportunistic citation.
