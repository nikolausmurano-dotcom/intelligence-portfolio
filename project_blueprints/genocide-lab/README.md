# Comparative Genocide Lab — comparative analysis of mass political violence across ten canonical dimensions

## Public Pitch

**What it is.** A comparative genocide analysis chamber built around ten canonical dimensions of mass political violence.

**Distinctive mechanism.** It uses six zones: Charter Wall, Zone II Dimensional Spine, Zone III Case Forge, Method Chamber with coding rules, Causal and Typological Observatory, and Campaign Table. The lab distinguishes concepts from operational rules, preserves the ten canonical analytic dimensions without contradiction, standardizes case cards, preserves contested-case protocols, supports first case-forge work, cross-case observatory analysis, and sprint and gate status, and is designed to match MuranoOS-style room logic through React/TypeScript and Rust/Tauri implementation paths.

**Why it matters.** It matters because comparative genocide analysis needs dimensions, cases, methods, evidence, process, and outputs linked together; otherwise cases become isolated moral descriptions rather than comparable analytical objects.
