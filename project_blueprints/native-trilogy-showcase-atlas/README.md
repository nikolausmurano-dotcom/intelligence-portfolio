# Book Trilogy Atlas

## Public Pitch

**What it is.** Book Trilogy Atlas is a manuscript-control atlas for a governed three-book inquiry into power, democratic formation, violence, memory, and repair.

**Distinctive mechanism.** It maps seventy-two chapters through canonical IDs, ReaderEdition targets, FinalProof argument backbone, ConstructionKits, PassageTargets, Democratic Dialectic voice material, bridge logic, dependency logic, terminology controls, chapter shape, chapter arcs, title state, visual identity, and rewrite governance across Public Book I, Public Book II, and Public Book III.

**Why it matters.** It matters because the trilogy's distinctive claim is architectural: power, democratic speech, and political repair have to be managed as one book system without flattening the three books into one voice or confusing public book order with canonical chapter IDs.

## Blueprint

### Vision

Book Trilogy Atlas is a manuscript-control atlas for a governed three-book inquiry into power, democratic formation, violence, memory, and repair.

### Mechanism

It maps seventy-two chapters through canonical IDs, ReaderEdition targets, FinalProof argument backbone, ConstructionKits, PassageTargets, Democratic Dialectic voice material, bridge logic, dependency logic, terminology controls, chapter shape, chapter arcs, title state, visual identity, and rewrite governance across Public Book I, Public Book II, and Public Book III.

### Built components

Built components visible or source-backed in this package: a public book-order map, canonical IDs, seventy-two chapter profiles, chapter types, chapter claims, carry-forward logic, reader-load/citation metadata where exposed, source-bound passage signals, governed rewrite-state metadata, priority source traceability, and a no-image Porkbun package that spends its size budget on native substance instead of generic artwork.

### Source use

Arendt and political theory support amor mundi, action, judgment, plurality, and the public world; security and intelligence literature supports Book I's restraint, secrecy, coercion, and analytic-integrity claims; peace/conflict and trauma literature supports Book III's repair mechanisms; Democratic Dialectic .docx supplies authorial cadence and conceptual ore, not paste-ready final prose.

### Plan

Execution plan: preserve canonical IDs; keep public order B2, B1, B3; upgrade each chapter by dossier rather than by freeform rewrite; classify every paragraph by rewrite mode; enforce book-specific register laws; run cross-book contamination checks; then replace public chapter prose only after evidence, citation, and connoisseurship review are complete.

### Status

Status is deliberately conservative. The chapter map, governance scaffolds, source traces, and rewrite dossiers exist as public-representable evidence of the project architecture. The governed rewrite-state file explicitly says live chapter prose has not yet been replaced and public prose admission is not complete; this page should not claim final manuscript completion.

### Remaining gaps

Remaining gaps: final title review, per-chapter source normalization, quotation placement, final public chapter prose, accessibility review, and full connoisseurship review for whether the prose reaches the standard of the strongest Democratic Dialectic passages without letting Book II's voice contaminate Book I or Book III.

## Maintenance Rule

Update this blueprint before strengthening public claims on the website. Do not use word count, module count, or status percentage as a substitute for concrete project substance.
