# Mashup Intelligence Room

## Public Pitch

**What it is.** Mashup Intelligence Room is a MuranoOS room handoff for audio-source management, phrase-level extraction, compatibility discovery, and studio workflow.

**Distinctive mechanism.** It routes audio sources through an Audio Resonance Room with a SQLite/LanceDB/DuckDB three-database architecture, Resonance Engine, AI Router Console, Automation Console, ingestion, phrase extraction, compatibility search, version memory, workflow state, and export planning so creative or analytic audio work can be managed as a structured room.

**Why it matters.** It shows the broader MuranoOS pattern: a domain is not just stored, but given instruments, state, search, and workflow controls appropriate to that domain.
