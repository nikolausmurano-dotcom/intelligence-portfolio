# Russia Security Desk

## Public Pitch

**What it is.** The Russia Desk is a two-page analytical workbench: a contemporary security desk for regime security, military adaptation, influence operations, Ukraine, NATO, and strategic decision-making; and a history page for the longer imperial, Soviet, post-Soviet, and European-order inheritance that gives present judgment depth.

**Distinctive mechanism.** The mechanism is source-lane to judgment-lane conversion. Internal workbench files, Kindle/Routledge/source library source families, journal and transcript manifests, marker packs, visual dossiers, exact-source queues, and source lineage bridges are separated by role.

**Why it matters.** It links regional knowledge to analytic method by preserving source lineage, strategic frame, adversary logic, and claim limits for later Russia/security work.

### Source use

Source families are assigned burdens rather than listed decoratively: strategic-culture and regime-security literature frames statecraft; Russia-Europe/NATO scholarship frames security architecture; Ukraine and military-analysis sources separate escalation, coercion, and alliance response; influence-operations sources prevent social-media reductionism; historical and archival reservoirs give the history page long-range depth; practitioner transcripts stress-test academic claims against operational vocabulary.
