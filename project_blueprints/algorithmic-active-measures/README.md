# Algorithmic Active Measures - data poisoning as counterintelligence tradecraft in AI-mediated intelligence analysis

## Public Pitch

**What it is.** A published-work surface on data poisoning as counterintelligence tradecraft in AI-mediated intelligence analysis.

**Distinctive mechanism.** It frames poisoned data as an operation against the foundations of analytic judgment, then uses a structured simulation to examine how coherent fabrications can deform AI-supported assessments.

**Why it matters.** It connects AI security to active-measures logic: the central issue is not only model error, but adversarial shaping of what analysts think the evidence says.
