# Counter-Terror Analysis — 13-page counter-terror analytical narrative arc

## Public Pitch

**What it is.** A counterterrorism and counterintelligence analytical arc, not a collection of detached topic pages.

**Distinctive mechanism.** The thirteen pages move through definition, radicalization pathway, institutional setting, adversary intelligence, planning cycle, capability conversion, vulnerability interruption, trust-capture disruption, deception defense, instructional diffusion, training interdiction, sustainment interdiction, target-confidence disruption, organizational structure, network concealment, civil-liberties lane discipline, defensive retrieval, intervention logic, and effectiveness review.

**Why it matters.** It demonstrates sequence reasoning: the value is in showing how CT/CI judgments develop across phases of capability, recruitment, concealment, and disruption rather than presenting isolated topic summaries.

### Built components

This is the most-recently-touched substantive work in the portfolio — the canonical 13-page doctrine package was iterating into its current form through the days immediately before the defense, and it remains the operational entry point when the page system is handed to an AI for ingestion or to a human for orientation. The depth of build varies deliberately across pages. Page 01 — the conversion-equation kernel that the rest of the arc rests on — has been built to its full readiness specification, with a parallel readiness audit, an explanatory spine, and a phase-gate protocol that takes a reader through the page at fidelity. Pages 02 through 13 render in the canonical visual mode and carry their full subroom content, but they do not yet have parallel high-fidelity treatment; the design choice was to demonstrate the per-page build at depth on the foundational page first, then extend. The arc has also pre-published the things it has not yet rendered well — source-confidence as visible visual certainty, the card-assembly motion in the retrieval page, the adversary-learning loop as a visible feedback structure rather than a static box, institutional side-effects, and the civil-liberties lane-discipline as a moving causal system. The current visual baseline is described against itself as a source-grounded baseline, not as a finished aesthetic.

### Source use

Source use is organized by role: method authorities, domain evidence, technical background, implementation evidence, and claim limits. The page should show why each source matters; direct quotations and stronger factual claims still require exact passage anchors before publication.
