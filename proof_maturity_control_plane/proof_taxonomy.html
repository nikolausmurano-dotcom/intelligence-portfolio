Proof taxonomy

Proof Levels

L1

Claim Scaffold

The project says what it intends to do, but the mechanism, artifact, or falsifier is still insufficient.

L2

Source-Linked Proof Scaffold

A source candidate is attached, but the proof still needs stronger artifact and test expression.

L3

Testable Proof Build

The artifact, operation, output, acceptance test, and falsifier are named.

L4

Instrumented Proof Path

The proof is tied to a project module, source-bound test, and claim ceiling.

L5

Execution-Ready Proof Capsule

The project has a reviewer-visible proof engine or runtime target capable of showing the mechanism.
