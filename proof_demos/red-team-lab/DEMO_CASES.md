# Proof Demo Cases: Red Team Lab — consolidated counter-deception lab across five authority roles

**Project ID.** `red-team-lab`

**Proof tier.** source-backed-proof-build

**Public claim.** A consolidated counter-deception lab for adversarial review of analytical work.

**Distinctive mechanism.** It treats red-teaming as a structured discipline with five authority roles: baseline canon, implementation packet, Prism Observatory, continuation pack, and bootstrap wrapper. The lab preserves six-table SQLite schema logic, canonical JSON project schema, replayable canon, patch-and-validation records, doctrinal core, command-registry deltas, cold-start re-entry, and a lineage appendix so a reader can see which authority supports which claim.
