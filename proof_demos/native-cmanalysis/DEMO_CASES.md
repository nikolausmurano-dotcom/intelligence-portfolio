# Proof Demo Cases: Collection Management Analysis

**Project ID.** `native-cmanalysis`

**Proof tier.** source-backed-proof-build

**Public claim.** A collection-management analysis page for turning intelligence needs into collection priorities.

**Distinctive mechanism.** It links requirements, priority intelligence requirements, source fit, collection gaps, reporting value, and feedback so collection is judged by decision utility rather than activity volume.
