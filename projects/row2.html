AI Rigor Detection - framework that detects performative rigor in AI-generated analysis; publication status is separated from validation status

A concise project summary for professional review, drawn from the maintained project record.

What this does

A rigor-detection framework for AI outputs that look scholarly, structured, or cited while failing to do the underlying evidentiary work.

Record note: project purpose

How it works

It separates the appearance of rigor from actual warrant: citation binding, source relevance, reasoning continuity, rubric fit, counterclaim handling, claim strength, Goodhart risk, adversarial-gaming flags, and validation limits are inspected as distinct failure surfaces.

Record note: working mechanism

Why it matters

It matters because AI-assisted analysis can fail by sounding careful; the project gives reviewers a vocabulary for detecting rigor theater and a path toward testing it against labeled rigorous and performative examples before it becomes accepted judgment.

Record note: professional value
