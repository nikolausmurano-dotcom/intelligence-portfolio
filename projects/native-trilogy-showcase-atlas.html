Book Trilogy Atlas

A concise project summary for professional review, drawn from the maintained project record.

What this does

Book Trilogy Atlas is a manuscript-control atlas for a method-bound three-book inquiry into power, democratic formation, violence, memory, and repair.

Record note: project purpose

How it works

It maps seventy-two chapters through canonical IDs, ReaderEdition targets, private support file argument backbone, ConstructionKits, PassageTargets, Democratic Dialectic voice material, bridge logic, dependency logic, terminology controls, chapter shape, chapter arcs, title state, visual identity, and rewrite method discipline across Public Book I, Public Book II, and Public Book III.

Record note: working mechanism

Why it matters

It matters because the trilogy's distinctive claim is architectural: power, democratic speech, and political repair have to be managed as one book system without flattening the three books into one voice or confusing public book order with canonical chapter IDs.

Record note: professional value

What is already there

Built components visible or source-backed in this package: a public book-order map, canonical IDs, seventy-two chapter profiles, chapter types, chapter claims, carry-forward logic, reader-load/citation metadata where exposed, source-bound passage signals, method-bound rewrite-state metadata, priority source traceability, and a no-image web host package that spends its size budget on native substance instead of generic artwork.

Record note: current status

Sources used

Arendt and political theory support amor mundi, action, judgment, plurality, and the public world; security and intelligence literature supports Book I's restraint, secrecy, coercion, and analytic-integrity claims; peace/conflict and trauma literature supports Book III's repair mechanisms; Democratic Dialectic .docx supplies authorial cadence and conceptual ore, not paste-ready final prose.

Record note: sources used

Still needed

Execution plan: preserve canonical IDs; keep public order B2, B1, B3; upgrade each chapter by dossier rather than by freeform rewrite; classify every paragraph by rewrite mode; enforce book-specific register laws; run cross-book contamination checks; then replace public chapter prose only after evidence, citation, and connoisseurship review are complete.
