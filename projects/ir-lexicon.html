IR Lexicon - coined-term lexicon for modern foreign-affairs discourse

A concise project summary for professional review, drawn from the maintained project record.

What this does

A coined-term lexicon for modern foreign-affairs discourse.

Record note: project purpose

How it works

It moves 400 raw coinages through a lacuna atlas, adoptability shortlist, 20-term canon, 10 mini-theories, 16-dimension adoption scoring, collision-risk flags, and a 25-rule propagation doctrine.

Record note: working mechanism

Terms such as Credibility Underwriting Gap, Compellence Overdraft, Ambiguity Rent, Sanction Saturation, Standards Entrenchment, Compliance Chokepoints, and Doctrine Lock-In are treated as analytic instruments rather than slogans.

Record note: working mechanism

Why it matters

It matters because IR language can become too vague to do analytic work; this project shows how a term earns use through definition, diagnostic power, adoption logic, and essay/briefing application.

Record note: professional value

What is already there

The lexicon is complete in the senses that matter for use: the canonical 20 are named with full schema, the lacuna atlas is filled, ten mini-theories are doctrinally elaborated, the 25-rule propagation doctrine is written, and the full 400-term corpus is preserved with the 16-dimension adoption scores.

Record note: current status

The package explicitly disclaims global trademark or originality clearance — collision-risk notes are internal heuristic judgments, not external IP screening.

Record note: current status

Sources used

Source use is organized by role: method authorities, domain evidence, technical background, implementation evidence, and claim limits.
