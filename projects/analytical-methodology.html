What this does

A Georgetown Applied Intelligence methodology pillar that treats eleven analytical disciplines as distinct working surfaces.

Record note: project purpose

How it works

It differentiates all-source, counterintelligence, crisis management, counter-operations, cyber, defense/diplomacy/intelligence, emerging-threats, foresight, homeland-security, intelligence-cycle, and public-service analysis by case anchor, source demand, method fit, output form, and cross-method composition.

Record note: working mechanism

Meridian Network, Operation GRAY DAWN, the Caracas Question, and HS-style signal-detection moves function as worked anchors rather than empty labels.

Record note: working mechanism

Why it matters

It matters because experts need to see which method belongs to which kind of problem, when one discipline can borrow from another, and how eleven adjacent pages become a working method library rather than a list of course topics.

Record note: professional value
