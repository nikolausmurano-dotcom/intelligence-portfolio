What this does

A consolidated counter-deception lab for adversarial review of analytical work.

Record note: project purpose

How it works

It treats red-teaming as a structured discipline with five authority roles: baseline canon, implementation packet, Prism Observatory, continuation pack, and bootstrap wrapper.

Record note: working mechanism

The lab preserves six-table SQLite schema logic, canonical JSON project schema, replayable canon, patch-and-validation records, doctrinal core, command-registry deltas, cold-start re-entry, and a lineage appendix so a reader can see which authority supports which claim.

Record note: working mechanism

Why it matters

It matters because adversarial review is not a posture; it is a system for preserving assumptions, sources, scope, contradictions, patches, doctrine, replay, and re-entry without forcing the public reader through dozens of internal handoff packages.

Record note: professional value
