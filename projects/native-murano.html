A concise project summary for professional review, drawn from the maintained project record.

What this does

A MuranoOS-linked manuscript command surface for coordinating the Trilogy and related human-agent work.

Record note: project purpose

How it works

It connects Tauri / React / Rust / SQLite desktop architecture with rooms, workstations, agents, commands, chapter architecture, reasoning matrices, continuity checks, collision checks, and revision state so manuscript development is treated as a method-bound operating environment.

Record note: working mechanism

Why it matters

It matters because long-form writing and agent-assisted development need local software, memory, roles, dependency checks, and review surfaces rather than isolated drafts.
