What this does

Prismatic Agora is a completed MuranoOS authority packet for a room-as-operating-system.

Record note: project purpose

How it works

It routes a room's purpose, roles, instruments, deliberation spaces, memory, method discipline rules, and export surfaces into one operating design so the room can host coordinated intellectual work.

Why it matters

It matters because collaborative knowledge work needs more than files: it needs a method-bound place where roles, evidence, disagreement, memory, and decisions can meet.
