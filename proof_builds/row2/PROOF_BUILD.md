# Proof Build: AI Rigor Detection - framework that detects performative rigor in AI-generated analysis; publication status is separated from validation status

**Project ID.** `row2`

**Proof tier.** source-backed-proof-build

**Public claim.** A rigor-detection framework for AI outputs that look scholarly, structured, or cited while failing to do the underlying evidentiary work.

**Distinctive mechanism.** It separates the appearance of rigor from actual warrant: citation binding, source relevance, reasoning continuity, rubric fit, counterclaim handling, claim strength, Goodhart risk, adversarial-gaming flags, and validation limits are inspected as distinct failure surfaces.

**Use value.** It matters because AI-assisted analysis can fail by sounding careful; the project gives reviewers a vocabulary for detecting rigor theater and a path toward testing it against labeled rigorous and performative examples before it becomes accepted judgment.
