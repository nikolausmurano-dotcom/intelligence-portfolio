# Proof Build: AR-SAT Room

**Project ID.** `mb-ar-sat-room-handoff-v0-1`

**Proof tier.** source-backed-proof-build

**Public claim.** AR-SAT Room is a design-and-schema handoff for an adversarially robust structured-analytic-technique room.

**Distinctive mechanism.** It routes an intelligence problem through a Room View that binds triad method, products, and proof; Idea Refinery, Patch Forge, baseline-comparison Evaluation Harness, JSON-Schema entities, command-palette behavior, station drawer, audit timeline, evidence handling, adversarial stress, and reviewer workflow make SAT use inspectable rather than decorative. The handoff names React/Next/Vite as the intended componentization path, while backend, auth, persistence, and vector search remain outside this pass.

**Use value.** It matters because structured analytic techniques become more useful when their inputs, adversarial failure modes, interface logic, data model, and review obligations are built into the workspace.

**Minimum public demo.** Run one analytic question through AR-SAT Room Handoff Identity; confirm evidence, assumptions, method decision, confidence limit, and human/AI delta.

**Claim ceiling.** The page may claim proof-structured components and visible verification targets. It may not claim external validation, production deployment, or completed empirical findings unless the project-specific record states those facts.
