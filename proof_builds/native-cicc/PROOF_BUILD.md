# Proof Build: ICC Complementarity Case Room

**Project ID.** `native-cicc`

**Proof tier.** source-backed-proof-build

**Public claim.** ICC Complementarity Case Room is a legal-threshold reasoning chamber for historical and operational case-study work on complementarity.

**Distinctive mechanism.** It turns ICC complementarity into a decision workflow: case definition, actors, constraints, causal mechanism, analogy limits, admissibility, domestic capacity, unwillingness, evidence burden, and jurisdictional claim limits.

**Use value.** It matters because complementarity is not only a doctrine label; it is a threshold test that decides when international jurisdiction should defer, intervene, or remain contested.
