Source utilization workbench

Sources As Load-Bearing Project Infrastructure

This layer makes source use operational. Each project receives a source strategy, role coverage, close-read queue, and component-level assignment. The purpose is not to display a bibliography; it is to show what each source is supposed to strengthen, what proof gate it affects, and what claim remains off-limits until passage-level reading is complete.

Claim Discipline

Metadata routing is useful for planning, source triage, and proof design. It is not citation support. A project may claim that a source has been routed to a role; it may not claim passage-level warrant until the close-read task is completed and recorded.
